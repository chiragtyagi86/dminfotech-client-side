# Installation

### Clone the Repository

```bash
git clone https://github.com/your-repo/dhanamitra-website.git
cd dhanamitra-website
```

---

# Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

# Environment Variables

Create a `.env` file in the backend directory.

Example:

```
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=dhanamitra
JWT_SECRET=your_secret_key
UPLOAD_DIR=uploads
```

---

# Run Development Server

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```

---

# Build for Production

### Frontend

```bash
npm run build
```

### Backend

```bash
npm start
```
